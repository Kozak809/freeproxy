# freeproxy

Использование: ./freeproxy <коды_стран> [опции]

Параметры:
  <коды_стран>        Код страны или список через запятую (ru, us,de,fr)

Опции поиска:
  -d, --deep <N>      Количество страниц для поиска (по умолчанию: 1)
  -r, --rating <N>    Топ N лучших серверов (по умолчанию: 1)
  -m, --max-latency <N> Максимальная задержка в ms (по умолчанию: без лимита)
  -p, --protocol <TYPE> Тип протокола: vless, vmess, trojan, all (по умолчанию: vless)

Режимы вывода:
  -c, --copy          Копировать первую ссылку в буфер обмена
  -o, --output <FILE> Сохранить ссылки в файл
  -v, --verbose       Подробная информация о серверах
  -i, --interactive   Интерактивный выбор сервера
  --qr                Показать QR-код для первой ссылки
  --random            Выбрать случайный из топ-10

Дополнительно:
  --test              Проверить доступность серверов (ping)
  --use-cache         Использовать кэш (быстрее, но может быть устаревшим)
  --update-cache      Обновить кэш серверов
  -h, --help          Показать эту справку

Примеры:
  ./freeproxy ru
  ./freeproxy ru -d 3 -r 5
  ./freeproxy ru,us,de -r 1
  ./freeproxy ru -m 100 -c
  ./freeproxy ru -d 2 -r 3 -o servers.txt
  ./freeproxy ru -i --qr
  ./freeproxy ru --random --test
